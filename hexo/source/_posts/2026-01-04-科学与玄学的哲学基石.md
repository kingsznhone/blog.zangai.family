# 何为科学

科学是一种系统化的知识体系和探索自然、社会及思维规律的方法。

它旨在通过观察、实验、推理和验证，揭示客观世界中的现象、本质及其相互关系，并在此基础上形成可检验、可重复、可预测的理论模型。

科学的核心特征包括：
- 经验性（Empiricism）：科学依赖于可观测、可测量的经验事实。一切科学结论都必须以实证为基础。
- 可检验性（Testability）：科学理论必须能够被实验或观察所检验。如果一个主张无法被证伪或验证，它通常不被视为科学。
- 可重复性（Reproducibility）：科学实验的结果应能在相同条件下被他人重复得出。
- 逻辑一致性（Logical Consistency）：科学理论内部以及与其他已被证实的理论之间应保持逻辑一致。
- 暂定性（Provisionality）：科学知识不是绝对真理，而是当前最佳解释；随着新证据出现，理论可能被修正甚至推翻。

现代科学建立在多种相互交织的哲学传统之上，主要包括：

#### 经验主义（Empiricism）
  知识源于感官经验；
  强调观察与实验是检验理论的最终标准；
  代表人物：弗朗西斯·培根、大卫·休谟、逻辑实证主义者。
  这是科学最根本的基石：没有经验证据，就没有科学。

#### 理性主义（Rationalism）
  强调逻辑、数学和先验推理在构建理论中的作用；
  例如爱因斯坦通过思想实验和数学推导预言引力波，远早于观测证实。
  科学不仅是“看”，更是“想”——理论常先于数据。

#### 可证伪性（Falsifiability）——卡尔·波普尔
  科学理论必须具有被经验事实潜在证伪的可能性；
  “所有天鹅都是白的”是可证伪的（只要找到一只黑天鹅）；
  不可证伪的主张（如某些形而上学或伪科学）不属于科学范畴。
  波普尔强调：科学不是“证实”，而是“未被证伪”。

#### 范式与科学革命——托马斯·库恩
  科学在“常规科学”时期围绕主导范式（paradigm）展开；
  当反常积累到一定程度，发生“科学革命”，范式转换（如从牛顿力学到相对论）；
  科学进步并非线性累积，而是非连续的、社会-历史过程。

#### 工具主义 vs 实在论
  工具主义：科学理论只是预测现象的工具，不必反映“真实”（如量子力学中的哥本哈根诠释）；
  科学实在论：成功的科学理论大致描述了世界的客观结构（如原子、电磁场真实存在）。
  这关系到我们如何看待科学“真理”的本质。

#### 贝叶斯确证理论（Bayesian Confirmation Theory）
  将科学确证视为信念度的合理更新，解释为何某些证据“支持”某个假说，被视为对归纳问题的一种现代回应。

## 科学的边界在哪里

### 哥本哈根诠释
哥本哈根诠释试图回答一个根本问题：量子力学的数学形式（如波函数）究竟描述了什么？我们该如何理解微观世界的“实在”？

尽管哥本哈根诠释长期主导教科书，但它面临诸多哲学和概念难题：

| 问题         | 说明                                           |
| ------------ | ---------------------------------------------- |
| 测量问题     | “测量”是什么？意识是否必要？仪器为何不量子化？ |
| 坍缩机制不明 | 坍缩是物理过程吗？如何与薛定谔方程协调？       |
| 主观性嫌疑   | 是否把“观测者”置于特殊地位？                   |

因此出现了其他诠释：

- 多世界诠释（Everett）：无坍缩，所有可能结果都在分支宇宙中实现；
- 玻姆力学（Bohmian Mechanics）：引入隐变量，恢复决定论；
- 退相干理论（Decoherence）：解释为何宏观世界看似经典，但不解决测量问题；
- 量子贝叶斯主义（QBism）：将概率视为主观信念，彻底放弃客观波函数。

所以哥本哈根诠释的核心在于不要问电子‘实际上’在哪里，而要问：在某种实验安排下，我们会观测到什么？

虽然人类对物质的探索已经深入到亚原子级别，但是存在的各种争议都聚焦于一个问题，我们如何看待这个世界的真实存在

### 量子贝叶斯主义
QBism（Quantum Bayesianism，量子贝叶斯主义） 是对量子力学的一种激进但逻辑自洽的诠释，它可以看作是对哥本哈根诠释核心精神的贝叶斯化重构——将量子理论从“关于世界的客观描述”转变为“关于主体信念的规范工具”。

1. 波函数 = 主观先验信念：
   - 波函数 ψ 不是“电子的状态”，而是我（作为代理）对“如果我对该系统做某测量，会得到什么结果”的置信度分配；
   - 它类似于贝叶斯统计中的先验概率分布，完全属于主体，不具有客观存在性；
   - 不同代理可以对同一系统持有不同波函数，就像两人对天气有不同预测。

2. 测量 = 获取经验，触发贝叶斯更新
    - 当我执行一次测量并得到结果，我就获得了新证据；
    - 于是，我按照广义的贝叶斯规则（通过量子形式的 Lüders 规则）更新我的波函数；
    - 这个“坍缩”不是物理过程，而是信念更新，正如你在抛硬币看到正面后，更新对硬币偏倚的信念。

3. Born 规则
    - P=∣⟨ϕ∣ψ⟩∣^2 在 QBism 中被理解为主体应如何根据当前信念（ψ）为其对特定测量结果（对应 ∣ϕ⟩）下注设定概率。
    - Fuchs 等人试图将 Born 规则从更基本的决策论原则中推导出来（如 Dutch book 论证），使其成为理性代理在量子世界中下注的“一致性约束”。

4. 测量问题
    - 因为根本没有“客观波函数”，所以不存在“何时坍缩”“谁算观测者”的难题；
    - 测量只是代理与世界互动的一个环节，结果是私人的经验；
    - 仪器、猫、朋友都可以是量子系统，只有当我接收到信息时，我才更新我的信念。在我打开盒子前，猫对我而言处于不确定状态（我的信念是叠加的）.打开后，我获得经验，更新信念——猫“对我而言”确定了生死。但这不意味着猫在客观上经历了坍缩。

QBism把量子力学从“自然之书”转变为“理性代理在不确定世界中行动的指南”。

在这个框架下，量子力学不是关于“世界是什么”，而是“我该如何期待我的未来经验”

科学不再是客观描绘实在，而是主体间通过共享经验与规则达成的实践协调。

## 我的科学信念

人猿相揖别，只几个石头磨过，小儿时节。人类对于客观世界的认知，随着知识的积累，技术的进步而发生改变。

在黑暗的中世纪，上帝创世论，地心说，四大元素，炼金术，在文明群体中拥有绝对至高的置信度。

同时在遥远的东方，阴阳，五行，天干地支，星宿节气主导着人们的生产与生活。

文明积累知识，一切都是从经验开始的。

一个理论如果被认为是正确的，都在一定程度上符合了当时的朴素生活经验与技术条件限制下可获得的最佳证据，所以文明群体对于某一理论的置信度就会达到一个较高的区间

生命对不确定性的厌恶是一种天性，随着文明与知识的传承，经验积累的越来越多，人类试图从海量的经验中总结出世界的规律。

几千年的前仆后继，一个又一个试图解释世界的理论被提出，人们再使用现实中的观测得到的经验进行验证，不断证伪错误的理论，由此产生了科学。

21世纪的此时此刻，人类对于数学物理化学的认识已经达到了相当的高度，理论体系在许多领域已高度成熟且预测精准，同时也不能不警惕几百年前科学界的自信陷阱。

人类对于物质世界的认识也许还处在非常幼稚的时期，也许现在看来的某个真理，会在很多年后被更强大更精确的实验更严密的计算推翻，出现更完善更自洽的理论，解释现有理论的漏洞。

科学的进步，往往始于少数人对主流共识的理性挑战，并在新证据面前实现共同体信念的更新。以往如此，未来也会如此。

我对于某个理论从来都不是深信不疑，而是会根据我所接受到的教育，群体的共识，私人的生活经验与美好愿景，共同凝聚成一个主观置信度。

#### 我对于质子中子电子的存在的置信度无限接近于100%

因为物质的组成与性质经过了无数科学家的验证，理论体系完备，如有必要我也可以自己动手做实验，得到理论预测的结果。

理论为人类生活的进步产生了重要指导。化学制药拯救生命，半导体制造成了芯片发展出互联网和AI，核物理带来近乎无尽的能源。

我多年所学所见，生活中所吃所用的一切，都和原子理论有非常紧密的联系。

#### 对于宇宙大爆炸理论，我的置信度是90%

根据现有天文学的观测，可以看到大爆炸后38万年的微波背景辐射。

根据理论物理计算，大爆炸后3分钟 -20分钟的原初核合成阶段，预测的氢、氦、氘、锂丰度与天文观测高度一致。

但是继续往前回溯到10⁻¹² 秒 – 1秒的电弱相变、夸克禁闭阶段，理论上是合理的，但在如此极端条件下的物质变化未完全验证

只有建造出比LHC更大的加速器才有可能去进行实验。

在大爆炸后的10⁻⁴³ 秒内，现有的一切物理理论都在如此极端环境下失效了，处于不可描述状态。

相比其他的宇宙生成理论，宇宙大爆炸理论的证据更多，所以它成为了当前主流的共识。

#### 对于虫洞的存在，我的置信度有50%

广义相对论的核心方程在某些特定的时空几何结构中，允许存在连接两个不同时空区域的喉部。

但是虫洞的关键问题在于稳定性，如果想要维持稳定性，就必须存在某种负能量密度的奇异物质，对这种物质性质的描述超过了我的理解能力，并且根据目前的实验水平很难找到这种物质。

但量子场论中的卡西米尔效应（Casimir Effect）表明，在微观尺度上确实可能存在负能量密度。

在个人愿景中，穿越到几光年外的行星上游玩一番，近距离观赏仙后座α星云散发出的七彩光芒，然后拍照发到朋友圈，难道不是一件很酷炫的事情吗？

所以我热切盼望未来能够产生出新的理论，实验的方法和设备能够不断进步，去探寻存只在于理论中的奇异物质。

#### 我对于多世界解释的置信度只有20%
对于多世界解释的置信度中，绝大部分是个人的美好愿景。

一个人的一生会有数不尽的遗憾和不甘，在无数个不眠深夜，都会思考一个问题，倘若那天做出了不同的选择，这一切会改变吗？

如果再努力一点，是不是结果就会不一样？平行世界如果真的存在，在做出选择的一瞬间，两个世界就再也不会相遇。

如此不可观测不可计算，但我还是愿意为了美好愿望中那微乎其微的可能性，选择相信一部分。
  
#### 对于地球是平的，我的置信度无限接近于0%

我曾亲自在三万英尺高空飞行，看到地平线呈现出优雅的弧度。

我看过猎鹰火箭发射的直播，地球从一个平面逐渐缩小成了美丽的蓝色弹珠。

通过数学计算，有了经纬度就能算出来应该朝哪里走，走多远就能到达目的地。

所以地平论在我看来无疑是可笑的闹剧。

回到最开始的问题，科学的边界在哪里？

我是量子贝叶斯主义者，我认为科学理论代表的是观察者对测量结果的主观信念，而不是客观实在。

在这个充满客观幻觉的时代，科学的边界在于测量设备和实验方法的性能极限，在于个人信念的置信度，在于人类作为一个文明整体对于认识世界的共识。

我相信的理论不是关于世界的本体论陈述，而是理性代理在不确定世界中进行决策的规范性框架。

科学的效力不来自它‘反映实在’，而来自它在不同主体在各自实践中的一致性、可协调性与预测成功率。


# 何为玄学 

广义的玄学包括人类对终极实在的理性与超理性探索的哲学，古代神秘学，以及预测技术。

反映了人类永恒的双重渴望，既要理解世界为何如此（哲学），也要知道明天该如何活（预测）。

## 哲思型玄学（形而上学/本体论探索）

*代表：《道德经》《庄子》、魏晋玄学、部分佛学（如华严宗“法界缘起”）、新柏拉图主义、斯宾诺莎泛神论等*

### 核心特征
- 追问存在的本源、宇宙的结构、道/理/气/梵的本质；
- 强调直觉体悟、超越语言逻辑（“道可道，非常道”）；
- 目标是安顿生命、获得智慧，而非预测吉凶；
- 具有高度理性思辨性（尤其在魏晋玄学和印度/希腊传统中）。

## 神秘学（Occultism）

*代表：赫尔墨斯主义、卡巴拉、炼金术、玫瑰十字会、通神学会、新时代运动（New Age）*

### 核心特征
- 相信存在隐藏的宇宙知识（gnosis），需通过秘传、启蒙或灵性修炼获得；
- 主张 **“如其在上，如其在下**（As above, so below）——宏观宇宙与微观人体同构；
- 强调象征、对应、能量、意识转化；
- 常融合宗教、哲学、早期科学（如炼金术是化学前身）。

## 预测学（Divinatory）

*代表：*

*中国：八字（四柱）、紫微斗数、风水（堪舆）、奇门遁甲、面相手相*

*西方：占星术（Astrology）、塔罗牌、灵数学（Numerology）、卢恩符文*

*其他：印度占星（Jyotish）、玛雅历法等*

### 核心特征 
- 基于符号系统与时间/空间模型，试图推演个人命运、事件吉凶、最佳行动时机；
- 依赖象征对应（如行星=心理原型，八卦=自然现象）；
- 强调个体与宇宙节律的同步性；
- 多为实用导向：“我该不该换工作？”“哪天结婚最吉利？”

## 广义玄学共同的底层逻辑

| 预设                       | 说明                                         |
| -------------------------- | -------------------------------------------- |
| 宇宙是意义整体             | 万物相互关联，无纯粹偶然；事件背后有隐藏模式 |
| 可见世界受不可见力量支配   | 如“气”“以太”“业力”“集体无意识”               |
| 人可通过特殊方法接入此秩序 | 冥想、占卜、仪式、直觉、数学推演             |
| 知识具有神圣性或危险性     | “天机不可泄露”，需师承或道德准备             |


人类文明发展过程中，随着科学与技术的不断进步，依然有一些事物无法被科学解释，从而发展出了玄学，作为解释这些事物的理论方向。

在中国玄学体系中，道德经与易经一般被称为道，八字紫薇奇门被称为术，道为术之体，术为道之用，就如同武侠小说中的内功与招式的关系。

古今中外的玄学系统，是否在用不同的“语言”描述同一套更深层的宇宙规律？东西方文明独立发展出了形式迥异但内核相似的占卜与命理系统，这强烈暗示了人类集体意识存在着某种共通的认知原型。

1. 符号映射的普适性
    - 无论是八字的天干地支、星盘的行星星座，还是塔罗的象征图像，本质上都是将抽象规律编码成符号系统。这类似于数学用公式描述物理世界。
    - 人类大脑擅长模式识别，这些系统可能抓住了某些概率模式或周期规律（如天体运行对地球生态/人类社会产生的周期性影响）。
2. 共时性原理的体现
   - 心理学家荣格提出的“共时性”认为，事件在时空中的“有意义巧合”可能揭示了某种超越因果的关联性。占卜行为本身，可能触发了意识与潜意识对这些隐藏关联的觉察。
3. 复杂系统的不确定性描述
   - 现代科学中的混沌理论、量子力学表明，世界并非完全决定论。玄学工具可能是一种处理复杂系统与不确定性的经验方法，类似于模糊数学或概率预测。

## 世界的潜在规律

正如同帛书道德经中所写，道可道也，非恒道也，各种玄学研究共同指向的潜在规律，是不是就是那个道？

各类玄学理论经历了文明几千年的哲学思辨探索与生活实践，是否在一定程度上代表了人类对这个世界某种深层次规律的局部认识？

有些玄学理论认为存在一个记录宇宙信息的“场”（如形态发生场、阿卡西记录），而不同玄学技术系统可能是局部接入这个场的“接收投影技术”。

量子贝叶斯主义认为现实由主体与世界的互动生成，观察者效应在量子层面已被证实。占卜或许反映了意识对现实某种程度的塑造或调频能力。

如果我们的世界是高维现实的投影，那么八字、星盘等可能是在用二维符号系统，描述多维信息在时空中展开的规律。

全息原理暗示局部包含整体信息，一个空间区域中所包含的全部信息，可以被完全编码在其边界表面上，描述自然的有效方式可能比我们想象的维度更低。

这种对世界的深层潜在规律，一个还未被现代科学所描述的规律的分析与思考，或许正是各类玄学研究所指出的方向。

## 我的玄学信念

### 认识这个世界的另一条路
我从小就不迷信，因为我小小的脑瓜想不通爸妈一些讲究背后的原理。

正月剃了头舅舅也没有死，晚上剪指甲也没有鬼来掐我的脖子。生活中的经历，把我对这些迷信的置信度降到了无限接近于0。

我一直相信，物质世界是有可预测性的，就如同忘了关火会把稀饭煮成锅巴，踹了一脚机箱会让硬盘划伤损坏。

人也是有可预测性的，在学校太调皮了老师会叫家长，下课以后玩太疯把书包弄丢了，妈妈会很生气，会拿毛衣签字抽俺的腚。

随着一年一年长大，数学和物理课给了我更多认识世界的工具，可知道的越多，就发现自己不知道的越多。

到了第一次喜欢一个女同学的年纪，我用各种笨拙的方法表达好感，可得到的却是尴尬又不失礼貌的拒绝。

我问了很多同学，想知道为什么她不喜欢我，每个人的说法都不一样，有说我气质猥琐的，有说我长得不好看的，有说我不会表达的，有说我不会哄妹妹开心的。让我印象最深刻的是一个同学被问烦了，给我的答案是：“不喜欢就是不喜欢，没有理由。” 我不甘心，我搜索一切能得到的知识，也没有得到一个让我满意的答案。这是我寻找这个世界规律的第一次失败。

又过了一年上了初二，我的身体飞速发育，脸型也从鹅蛋变长，大脑接收信息和处理信息的速度也获得了极大的提升。

我想了解我遇到的每一个人的遭遇与所想，以及背后深层次的原因，包括我自己。可我发现课本上学到的知识只能解决我无数困惑中的一小部分，意识到人是最复杂的存在。

那年IPhone 4发布，看到一个同学下课拿出全面屏的Iphone 4玩游戏，而我的兜里揣着一个充话费送的直板山寨机，甚至都不是诺基亚，功能只有打电话发短信。

为什么他能拥有Iphone 4而我不能？因为妈妈不给我买。

为什么爸妈不愿意给我买？因为妈妈没有很多钱。

为什么妈妈没多少钱？因为她的收入和支出决定了没有太多盈余。

为什么妈妈没有一个收入更高的工作？因为她读的大学以后被分配到了现在的工作上。

为什么妈妈能去读大学？因为高中的时候，贵州大山里的军工厂里有一批上山下乡的清北知青来支教，加上个人的努力，考上了80年代的大学。

为什么妈妈会在贵州深山的军工厂长大？因为60年代中苏交恶，军工厂从东北搬迁到了大山深处

一环一环思考下去我意识到，我在当下的现状，与无数历史中的人与事产生了联系。

于是我开始彻夜不眠，去思考自己这十几年来所见的各种事情背后因果链条，并试图从中寻找规律。

可无论我如何思考，都还在困惑，因为有太多的未解之谜。

为什么一个人在某个时间，做出了这样的决定，而没有选择另一条道路？

为什么某件事情会在这一刻发生，而不是在下一刻，或者干脆不发生。

偶然有一天，在妈妈的书架上看到一本《水镜神相》。我的第一反应是迷信书籍，我倒要看看这书上写的是什么怪东西。

书上说嘴唇薄的人能言善辩，嘴唇厚的人不善言辞。我去上学，观察身边的人，发现老师们的嘴唇都很薄，讲课骂人的时候滔滔不绝，大道理一套又一套。唯一一个厚嘴唇的老师，平时只会慢悠悠念课本，骂人的时候也只会说别给我操蛋。

此时我还不知道这个东西叫玄学，经过观察身边的人，我对这本书的置信度上升了。

面相学给我提供了一个截然不同的认识世界的工具，这让我产生了极大兴趣。

随着学习的深入，我不断观察身边人的样貌，并与书上所学的知识进行交叉对比，并以此给一个人进行人格画像，每次正确的判断，都在提升我对这本书的置信度。

很多年过去了，虽然忘了书上的论点和公式，但是看到一个人的脸，就能大体判断出这个人是好人还是坏人，模糊的性格和特质，值不值得沟通和信赖。

相学理论也有预测失败的时候，但是在大量样本的输入下，潜意识里似乎产生了一个logit尺度上的校准模型，模型的输入输出不断更新着我对于面相学的置信度。

现在我突然领悟，我当年是在用面相书上的数据标注，训练自己的视觉神经网络。面相从本质上是一门统计学，是古人通过大量的观察和验证，总结出来的关于人的样貌与性格特质的相关联的一套数据集。

原来这个世界还能被另一套理论所描述，玄之又玄，众妙之门。

### 打开新世界的大门

有了面相学的引路，我对玄学的探索开始了，我第一个想到的就是道德经，毕竟在文艺作品中，玄学和道法经常一起出现。

我天真地以为能在道德经里看到非常厉害的理论，能让我进一步理解这个世界。可我打开道德经的时候我傻眼了，每个字我都认识，怎么连起来我就看不懂了。

于是我只能用浅薄的人生阅历尝试理解道德经里所说的“道”是什么，通篇读完，脑袋空空，只记住了道可道，非常道。

也许对世界本质的探索是一条永远走不完的路，以我现在的能力不足以摸到一分一毫，所以我认输了。

因为整晚整晚不睡觉，导致我的精神状态出了很大的问题，和妈妈关系也非常紧张。

在无数争吵与斗气之后，百般无奈下，我妈带着我去找了一个老道。老道在看了我的八字以后，说我最好改个名，能够改善我的情况，于是我便改了名。

改了名以后，我的心境变得平和下来，不再和妈妈吵架，歇斯底里也变成了安静如鸡，我的人生仿佛被按下了重启键。

过了一年半载，回顾自身发生的变化，我发现最大的变量只有改名。

如果名字就能对一个人的命运产生如此巨大的影响，那么相较于个体对自我的规划和努力，名字又是通过什么作用影响着一个人的命运呢？

我想的好累，学业繁忙让我没有更多时间去思考诸如此类的哲学问题，我对玄学理论依然保持怀疑，但它在我心中埋下了一颗种子。

### 向更深处探寻

弹指一挥间，我上了大学，来到了美国。目睹同一个宿舍的同学因为航校的种种恶劣行径，停飞回国转专业，在驰骋于蓝天前的最后一步戛然而止。

我恐惧，不知道我是否也会在未来的某一天灰头土脸回国转专业，无颜面对江东父老，每天都在问自己，明天会怎么样？

每天晚上一个人在宿舍门口，一根接着一根抽，回想到大一大二的军训，集合，跑操，悬梯，跑五千米流的汗。

想到自己因为对未来不确定性的迷茫，放弃了考一个名牌大学，选择把自己的狗命卖给航空公司，换取铁饭碗的确定性。

我渴望确定性，所以选择了看似最确定的一条路。可身处两万公里外的异国他乡，我面对的依然是看不清未来的重重迷雾。

我想起了那天看到老道认真细致地查询干支表，在A4纸上对着我的八字逐字推演计算。心里的种子在恐惧和迷茫的浇灌下发芽了

于是我开始学习最基础的阴阳五行天干地支，在互联网上收集碎片化的知识，但碎片化的知识无法形成足够让我理解的体系，所以我向我最不擅长的语文发起了冲锋

我开始啃古籍原著，《渊海子平》《子平真诠》《三命通会》《滴天髓》，先从囫囵吞枣开始，记住书中的定理与公式。

接着在列表里抓同学朋友当小白鼠，照本宣科分析一番，大胆假设，小心求证，遇到错误就回到书中寻找答案。

一开始学习就像是在瞎蒙乱猜，我也怀疑过这个东西是坑蒙拐骗的伪理论。

坚持学习几个月以后，我初步察觉到有的时候根据理论计算，能通过理论计算出一些瞎蒙都不敢给出的结论，竟能完美对应现实情况。

身边认识的小白鼠用完了，我就注册了匿名交友APP，复制粘贴一个免费算命文案，天天抓小白鼠做实验。

通过在无数网络小白鼠身上练手，结合每天的复盘，思考和学习，我的水平日渐提高。

仅凭出生日期，就可以得到关于一个陌生人如此多的信息，对某些八字组合推算出的多个方面现实情况表现出了惊人的似然度，这也震撼到了我自己。

年仅6岁的我，在贝叶斯信念的框架下，从爸妈生活中迷信得到的先验知识，对于玄学的置信度无限接近于0。

通过持续二十多年收集后验知识进行修正，如今我对于玄学的置信度还在实践中不断提高。

### 之谓道

学习到一定水平后，我开始思考更本质的问题，为什么这些预测学都在一定条件下显现出独特的有效性？

没有任何自然科学理论能够解释命理学的原理，命理书上写的公式与算法，只能把它当作玄学体系内的公理去使用。

根据玄学公理和计算法则进行计算，算的是某个现实中事件的发生概率，而在写出断言的时候，概率被二元化表达出来。

在宏观世界中一件事情的发生与不发生，本身就是二元对立的，就如同已被观测的波函数坍缩成一个确定的结果。

计算者根据自己的知识和计算水平得出的内心概率极难被量化，而二元化的断言就一定会在某一次验证时产生与现实情况不符合的错误。

计算者的理论水平决定预测概率的校准与区分度，而且有利益相关者通过设计话术套用玄学概念进行坑蒙拐骗，所以玄学的置信度在人群中有非常大的分布区间。

传世的玄学工具理论体系或许也不够完善，用一个不够完备的理论进行计算，得出的结果与真实也会有偏差。

我认为，一个如“道”所描述的、宇宙固有的、深刻而统一的终极深层规律极有可能存在，而《道德经》中的“道”，是人类对它的最精妙、最富启发性的描述之一。

八字、星盘、塔罗等分析技术在我看来，更像是人类在不同文化背景下，试图测绘或解读这个“道”在现实中可能产生的“波纹”或“投影” 。

它们用的是不同的符号语言和映射规则，对过去、现在和未来进行计算和分析，而“道”本身，是那个产生变化的根本动力。

所有工具都可能触及了“道”的某些局部或表象规律（如周期、关联、对立转化），但任何工具都无法等同于“道”本身。

为道日损，过于执着于复杂的推算工具，有时反而可能远离了对根本规律的直觉把握。

现有的科学，是探索这个“道”在可观测、可量化的维度上（主要表现为物质与能量的规律）最精确的工具。科学的符号系统，主要揭示的是世界稳定、重复的外在表现规律。

而东西方的古老玄学，则试图从另外的视角，对那些难以量化、更具关联与意义的维度进行直觉性描绘和隐喻性表达。玄学的符号系统，更多地指向一种关于世界动态、关联的内在隐含规律的猜想。

两者并非对立，而是探索同一座“冰山”的不同路径，科学精细地测量水上的部分，而“道”则提醒我们水下那更庞大、更根本的整体的存在。
